🔑 闲鱼Cookie配置

首次使用需要配置Cookie，仅需一次，有效期约24小时。

📋 获取Cookie步骤

  1. 打开 Chrome 浏览器，访问 goofish.com 并登录
  2. 再打开一个新标签页访问 taobao.com（确保已登录）
  3. 按 F12 打开开发者工具 → 切换到 Network 标签
  4. 刷新页面，点击第一个请求，在 Request Headers 中找到 Cookie: 开头的一整行
  5. 右键 → Copy value → 粘贴到下方文本框

或者安装浏览器扩展 "EditThisCookie" 直接导出所有Cookie
